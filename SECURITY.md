# Security Policy

## Supported Versions

Only the latest version of Nimble is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities to [chesssoumalya21@gmail.com](mailto:chesssoumalya21@gmail.com). 

We will acknowledge your report within 48 hours and provide a timeline for a fix. Do not disclose vulnerabilities publicly until a patch has been released.
